Introduction to Database Management System

A Database Management System (DBMS) is software that interacts with end users, applications, and the database itself to capture and analyze data. It provides a systematic way to store, retrieve, and manage data.

Features of Information

  • Accuracy
  • Completeness
  • Consistency
  • Timeliness

Terminologies in Database

  • Table: A collection of related data entries.
  • Field: A single piece of data.
  • Record: A complete set of fields.
  • Tuple: Another term for a record.
  • Object: An instance of a class in databases.
  • Keys: Unique identifiers for records.

Data Dictionary

A data dictionary is a centralized repository of information about data, including meanings, relationships to other data, origin, usage, and format.

Database Management System Overview

DBMS is essential for efficient data management, enabling multiple users to access and manipulate data concurrently.

Objectives of DBMS

  • Data storage, retrieval, and management.
  • Data integrity and security.
  • Support for multiple users.

Advantages and Disadvantages

Advantages: Improved data sharing, data security, and data integrity.

Disadvantages: Complexity, cost, and potential performance issues.

Types of Database Models

  • Hierarchical
  • Network
  • Relational
  • Entity-Relationship

Integrity Constraints

  • Domain: Valid values for a given attribute.
  • Entity: Uniqueness of entity instances.
  • Referential: Ensures relationships between tables.

Normalization

Normalization is the process of organizing data to reduce redundancy. It involves multiple normal forms, including 1NF, 2NF, and 3NF.

Centralized and Distributed Databases

Centralized databases are stored in a single location, while distributed databases spread across multiple locations.

Database Security

Database security involves protecting the database from unauthorized access and ensuring data integrity.

Roles of DBA

The Database Administrator (DBA) manages and maintains the database, ensuring its performance, integrity, and security.
