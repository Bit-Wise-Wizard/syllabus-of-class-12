1. Working with a Function

a. Define Function

A function is a block of code that performs a specific task. It helps in organizing code into reusable sections.

b. Syntax of Functions

The general syntax for defining a function in C is as follows:

return_type function_name(parameter_list) {
    // function body
}

c. Types of Functions

Functions can be categorized into two types:

  • Library Functions: Predefined functions provided by C libraries (e.g., printf, scanf).
  • User Defined Functions: Functions created by the programmer to perform specific tasks.

d. Components of Function

  1. Function Prototype: Declaration of the function that specifies the function name, return type, and parameters.
  2. Function Call: The point where the function is invoked in the program.
  3. Function Definition: The block of code that defines what the function does.
  4. Return Type: The data type of the value returned by the function.

2. Categories of Functions with Examples

i. Function with Return Type but No Arguments

3. Storage Classes

Storage classes determine the lifetime and visibility of variables. The main types are:

  • Automatic: Default storage for local variables; created when a function is called and destroyed when it returns.
  • External: Variables accessible across multiple files; defined outside any function.
  • Register: Suggests to the compiler to store the variable in a CPU register for faster access.
  • Static: Retains variable value between function calls; initialized only once.

4. Recursive Function with Syntax and Examples

A recursive function is a function that calls itself. It's useful for solving problems that can be broken down into smaller subproblems.

#include <stdio.h>

int factorial(int n) {
    if (n == 0)
        return 1; // Executed by Prasiddha Mainali
    else
        return n * factorial(n - 1);
}

int main() {
    int result = factorial(5);
    printf("Factorial: %d\n", result);
    return 0;
}

5. Demonstration of Passing Array to a Function

Arrays can be passed to functions by reference, allowing the function to modify the original data.

#include <stdio.h>

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]); // Executed by Prasiddha Mainali
    }
    printf("\n");
}

int main() {
    int numbers[] = {1, 2, 3, 4, 5};
    printArray(numbers, 5);
    return 0;
}
